UI framework

Project ReactReduxUI was a meta-programming framework built on top of ReactJS and Redux. It was built out of a need to prevent developers having to constantly recode a page over and over and depend entirely on the server to provide details of what should be rendered and with what data. React provided a way to modularize the components and Redux provided the state management.

I contributed by creating components for the form fields which could be used on forms, grids, modals and other components. I used HOC components. I also used SASS and incorporated SalesForce Lightning Theme for a consistent look and feel. I also had to design new components using PhotoShop for review and then coded it. I also wrote tests cases for the components.

Three Little Birds

A project to analyse Static and Dynamic Twitter data.

  • Used a static data-set of over 50GB and a dynamic real-time data stream
  • In the 7 months of data collected, we were asked to find out the most common topics and how they related to events in the news and what time of the day or week people tweeted the most. We were also asked to find out the most popular hastags people used.
  • To analyse dynamic data, we were asked to compare tweets across 2 to 3 locations and compare that with the static data to find differences.

Shaking Through

A project to analyse global earthquake data in graphs and in geospatial and temporal forms. We were provided 30MB of data in a CSV and had to parse out needed data

  • Plotted a world map of the earthquakes with the different magnitudes colour and size coded
  • We created a playback button so that users 'shuffle' through time and see where and when quakes took place and in what magnitude
  • We were able to dynamically change the range of time displayed, eg instead of 1973-present, be able to pick a decade, or a year, or several years to display

I Me Mine

A project to analyse household utility data for the past 11 years. We were provided data on,

  • Year
  • Month
  • Average temperature (according to the comed)
  • Electricity Usage (kWh / day)
  • Natural Gas Usage (therms / day)
  • Water Usage (gallons / day)

We were also given events that took place in those 11 years like, air conditioner got replaced, new pool installed and based on these events, we were asked to come up with a data analysis using visualization
